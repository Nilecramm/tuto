Rechercher dans le titre / le texte / l'url:

Pour rechercher un mot dans le titre, le texte ou encore l'url d'un site:


Pour rechercher dans le texte, nous pouvons utiliser le préfixe "intext:", ou "allintext:" pour plusieurs mots, par exemple, si l'on tape: allintext:wikipedia.org pomme fleur, google cherchera sur le site Wikipédia une page qui comporte les mots pomme et fleur.

Pour rechercher dans un titre, nous pouvons utiliser les préfixes "intitle:" ou "allintitle:", par exemple, quand l'on tape dans google: intitle:pomme, on obtiendra des sites avec le mot pomme dans le titre

Pour rechercher dans une url, nous pouvons utiliser les préfixes "inurl:" ou "allinurl:", par exemple, avec la recherche: inurl:fleur, google nous renverras à des sites avec le mot fleur dans leur url

Le préfixe "allin" permet de réunir plusieurs mots mais on pourrait aussi écrire avec plusieurs préfixe en "in", par exemple: intitle:chaise intitle:bois = allintitle:chaise bois
